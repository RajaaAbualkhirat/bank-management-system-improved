# Banking Management System

This project is an improved version of the original project:

Original repository:
https://github.com/harshitbansal373/Simple-bank-managemnet-system

The original code was used as a starting point, and several improvements were applied based on Software Construction principles.

## Improvements

- Better variable and function naming.
- Improved error handling.
- Removed magic numbers.
- Applied DRY principle.
- Reduced code duplication.
- Replaced recursive menu calls with loop-based control.
- Improved maintainability and reliability.

This project was developed for educational purposes.
